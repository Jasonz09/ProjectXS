# ProjectXS Competitive Rank System

## Overview
ProjectXS features a unique 10-tier competitive ranking system. Players earn **Rank Rating (RR)** points to progress through ranks, with each rank requiring 100 RR to advance.

---

## Rank Progression System

### Earning RR
- **Win**: +15-25 RR (based on performance)
- **Loss**: -10-20 RR (based on performance)
- **Draw**: ±5 RR

### Rank Advancement
- Earn **100 RR** to promote to next rank
- RR resets to 0 upon promotion
- Cannot demote below 0 RR in current rank (demotion protection)

### Performance Factors
RR gains/losses are influenced by:
- Match outcome (Win/Loss/Draw)
- Individual performance (K/D/A ratio)
- Combat score
- Objective participation
- MVP stars

## Unique Features

### Why These Names?
Unlike generic metal-based rankings, ProjectXS ranks reflect:
- **Tactical progression**: Initiate → Operator → Specialist
- **Combat excellence**: Elite → Vanguard → Phantom
- **Legendary status**: Eclipse → Apex → Sovereign → Nexus

### Thematic Alignment
- Fits sci-fi tactical shooter aesthetic
- Reflects character roles (Challenger, Controller, etc.)
- Emphasizes skill progression and mastery
- Creates unique ProjectXS identity
